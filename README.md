# 🌟 About CartIQ

**CartIQ** is a full-stack retail billing and inventory management system developed to simplify retail store operations through automation and smart digital workflows.

The project combines:

* ⚡ Fast & Secure Backend APIs
* 🎨 Responsive Modern UI
* 🔐 JWT-Based Authentication
* 📦 Inventory & Product Management
* 💳 Online Payment Integration
* ☁️ Image Upload & Storage Support

# ⚙️ Backend Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/CartIQ.git
```

---

## 2️⃣ Configure Database

Update `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/cartiq
spring.datasource.username=root
spring.datasource.password=yourpassword
```

---

## 3️⃣ Run Backend

```bash
mvn spring-boot:run
```

Backend runs at:

```bash
http://localhost:8080
```

---

# 🎨 Frontend Setup

```bash
npm install
npm start
```

Frontend runs at:

```bash
http://localhost:3000
```
